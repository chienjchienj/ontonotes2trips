install

1.1: set up or prepare for use

  • We installed a new furnace in the basement.
  • Will you help me install this software?
  • The plumber is coming tomorrow to install the new washing machine.
  • They installed the rule to protect students.

  • We will hold a ceremony to install the new governor.
  • She has installed a couple of young academics as her advisers.
  • She was installed in the English Department.

  • I've installed myself in the spare room.
